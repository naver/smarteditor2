SmartEditor Basic 배포본

SmartEdtitor는 Javascript로 구현된 웹 기반의 WYSIWYG 에디터입니다. SmartEdtitor는 WYSIWYG 모드 및 HTML 편집 모드 제공, 자유로운 폰트 크기 설정 기능, 줄 간격 설정 기능, 단어 찾기/바꾸기 기능 등 편집에 필요한 다양한 기능을 제공하므로 사용자들은 SmartEdtitor를 사용하여 쉽고 편리하게 원하는 글을 작성할 수 있습니다. 그리고 에디터의 구조가 쉽게 기능을 추가할 수 있는 구조로 되어 있어 SmartEdtitor에 기능을 추가하기 위해서는 정해진 규칙에 따라 플러그 인을 만들어 추가하기만 하면 됩니다.

현재 SmartEdtitor는 네이버, 한게임 등 NHN의 주요 서비스에 적용되어 있습니다. 그리고 Internet Explorer 6 이상, FireFox 2 이상, Safari 3, Opera 9, Chrome등 다양한 브라우저를 지원하고 있으며 지속적으로 지원 대상 브라우저를 넓혀 갈 예정입니다. 또한 지속적인 기능 추가를 통해 편리하고 강력한 에디터로 거듭날 것입니다. 

라이센스 : LGPL v2 
홈페이지 : http://dev.naver.com/projects/smarteditor

배포본에 포함된 파일은 다음과 같습니다.

/css : css 파일
/js : 적용시에 사용 하시는 js 파일이 들어 있습니다.
/js_src : 소스 수정시에 사용 하실 수 있는 개별 파일들이 들어있습니다.
install_guide.html : 설치 가이드
readme.txt : 간략 설명
release_notes.txt : 릴리즈 노트
sample.php : SEditorDemo.html을 이용해 편집한 내용을 서버에서 받는 php 예제 입니다.
se_blank.html : 에디터의 편집 영역을 나타내는 파일으로 에디터 설치시에 반드시 필요한 파일 입니다.
SEditorDemo.html : 에디터 데모 페이지 입니다. 적용 시에도 참고 하실 수 있습니다.
SEditorSkin.html : SEditorDemo.html에서 로드 하는 에디터의 스킨(디자인) 파일 입니다.
src_include.txt : 개별 소스를 수정 하고자 할 경우 편리하게 copy/paste 하실 수 있는 코드 입니다.
HuskyMessageMap.html : 소스를 이용해서 메시지 흐름 테이블을 생성 해 주는 파일 입니다.
HuskyMessageMap.css : HuskyMessageMap.html에서 사용 되는 css 파일 입니다.
projHelper.php : js 파일을 하나로 합쳐 주는 스크립트 입니다.
projHelper.ini : projHelper.php의 설정 값으로 어떤 파일들을 무슨 이름으로 합칠지 설정 해 놓은 파일 입니다.
